# Contributing to is-a.mbbs

## Pull Requests

- Follow the PR template exactly. PRs that don't follow the template will be closed.
- One subdomain per PR unless discussed with maintainers.
- We reserve the right to reject any PR for any reason.
- If changes are requested, make them promptly — PRs idle for 7+ days may be closed.

## Issues

| Reason | Label to use |
|--------|-------------|
| Report an abusive or harmful subdomain | `report-abuse` |
| Problem with your own domain | `support` |
| Feature suggestion | `suggestion` |
| General question | `question` |
| Anything else | choose the closest label |

## Code of Conduct

Be respectful. This is a community for medical professionals and students.
Medical misinformation, harassment, and spam will result in an immediate ban.
